单向链表与双向链表

1、理解python中交换变量语法a,b = b,a

python中变量标识（如 a = 10）维护的是一个地址，对其赋值实际上是改变该地址==指向的对象==。

2、单向链表的概念

链表由多个节点组成，每个节点包含两个信息（域）：当前节点元素（信息域）和下一节点地址（链接域）。链接指向链表中下一节点，最后一个节点的链接域指向空值。

5、双向链表

双向链表相比单链表，每个结点多了一个prev信息，指向前一结点。在实现方法时只需要重写add() append() remove() insert()，其他可以继承单链表。
